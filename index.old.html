Robust API hosting included

In order to power your visualisation or reporting application we'll produce and host a powerful web API (a way for computers to talk to each other).

Depending on your needs, you might like to make this available to third parties so that others could write computer programs that also take advantage of your data, furthering your impact.

Features

  • Update the underlying data from Excel, Google Sheet or other software using our intuitive interface
  • Fast, robust data server makes your data available to the interactive visualisation
  • Embed the interactive visualisation anywhere you like - on the web, on mobile, desktop or tablet
